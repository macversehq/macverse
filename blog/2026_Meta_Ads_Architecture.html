Execution Blueprint

Step-by-Step Campaign Setup

How to structure your Meta Ad Account to run autonomously with high conversion efficiency.

Step 1

Pixel and CAPI Setup (The Foundation)

Establish first-party, server-to-server tracking. Push Purchase Event Match Quality above 8.0 by sending email, phone, and IP parameters.

Step 2

Design the Creative Testing Sandbox (Manual Campaign)

Build a standard, manual (non-ASC) campaign using Campaign Budget Optimization (CBO) and Broad Targeting. Activate Dynamic Creative to run the 3:2:1 matrix.

Step 3

Launch the Scaling Engine (Advantage+ Shopping Campaign)

Consolidate scaling budget inside ASC. Set a strict customer budget cap of 5-10% under Controls to target pure acquisition prospects.

Step 4

Post ID Graduation (Creative Migration)

Locate the winning dynamic creative's Post ID from Page Posts manager. Insert the exact Post ID directly to ASC to keep all social proof intact.
